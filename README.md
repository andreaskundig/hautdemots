copule, ongulé
==============

J'ai fait un programme simple qui retourne beaucoup de résultats un peu répétitifs. Il tourne dans ton browser quand tu ouvres mots.html.

Dans mots.html, le javascript commence par une variable lettres_equivalentes ou tu peux regrouper les lettres différement. Je ne suis pas sur que c'est une bonne idée de mettre e avec é par exemple, les gens risquent de ne pas comprendre ce que c'est qu'un ongule.

La liste de mots se trouve dans mots.js.

Je l'ai trouvée ici http://www.pallier.org/ressources/dicofr/dicofr.html , mais elle ne m'a pas l'air d'être très sérieuse: il manque les gros mots.
